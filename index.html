Final Project Proposal

Product Name: Flyer's Forum

Team Memebers: Jack Tegeder

Application Overview: A HTML webpage that’s hosted by Microsoft azure that displays tips and tricks that college students can use to help fight anxiety and depression. The webpage will have stories from college students including their personal story’s regarding anxiety and depression, and how they overcame it. It will also include tips and tricks from certified psychiatrists and counselors.

Technology Overview:The website will be made in visual studio code, and will use Python version 3 as the language. It will be hosted using Microsoft Azure. It will be available to view on a standard google search browser, and possibly for a mobile device.
